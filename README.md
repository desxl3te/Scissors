**Scissors Bar** — это не просто сайт, это цифровая экосистема для самого атмосферного заведения в городе. Мы создали платформу, которая сочетает в себе эстетику индастриала, толику сарказма и глубокое понимание локального комьюнити. 

> *Наш бар — как хорошие ножницы: два острых лезвия, работающих в идеальном тандеме. И да, мы знаем, о чем вы подумали.*

---

## 📖 О проекте

Сайт разработан для автоматизации взаимодействия с гостями и создания уникального пользовательского опыта. Мы ушли от скучных ресторанных шаблонов в сторону "зумерского" дизайна. Здесь каждый элемент — от названий кнопок до анимации загрузки — содержит тонкие отсылки к лесби-культуре и современному юмору.

---

## 🛠 Функциональные возможности

### 1. Интерактивная Карта Напитков "The Cut List"
* **Динамическое меню:** Фильтрация коктейлей по "уровню драмы" и крепости.
* **Секретные позиции:** Если кликнуть на иконку ножниц, открывается доступ к шотам, которых нет в основном меню.
* **Состав с подтекстом:** Например, коктейль *«Broken Vows»* (со вкусом стекла).

### 2. Система Бронирования "U-Haul Express"
* **Быстрое бронирование:** Для тех, кто готов переехать за столик бара уже через 5 минут после знакомства.
* **Выбор зоны:** "У окна" (для тех, кто ждет бывшую), "У бара" (для тех, кто ищет будущую) или "Темный угол" (для интровертов).

### 3. Календарь событий "Events & Red Flags"
* Интеграция для анонса тематических вечеров

### 4. Личный кабинет "The Roommate"
* Система лояльности: копите "баллы за верность" и обменивайте их на бесплатные закуски.

## 🎨 Технологический стек

| Технология | Назначение |
| :--- | :--- |
| **HTML5 + CSS + JS** | Святая троица фронтенда. База, на которой держится всё — надежная, как твоя любимая фланелевка. |
| **Flask** | Микрофреймворк на Python для основной логики. Гибкий и лаконичный, когда не хочется связывать себя тяжелыми обязательствами. |
| **FastAPI** | Асинхронный бэкенд для мгновенных запросов. Реагирует быстрее, чем ты на уведомление от «той самой» в 2 часа ночи. |
| **MySQL** | Реляционная база данных. Помнит все твои брони и предпочтения лучше, чем твой личный дневник. |
| **GSAP (GreenSock)** | Инструментарий для продвинутой анимации. Делает интерфейс таким же плавным, как твой первый танец на барной стойке. |
